------------ FR : ---------------!


Le dossier contient les fichiers permettant de faire tourner le programme correctement.
Le programme principal du jeu est le fichier nommé : `Jeu_DIAMOND.py`. Le dossier contient également un fichier de test pour certaines fonctions, un dossier
contenant toutes les images que nécessite l'interface graphique, et un fichier .py nommé tkiteasy permettant de faire fonctionner l'interface graphique.

Le but de ce programme est de jouer au jeu appelé DIAMOND dans lequel les joueurs doivent choisir à chaque tour entre rester dans l'expédition ou partir.
Tant que les joueurs restent, les gains obtenus pendant la manche ne sont pas validés. Ils doivent donc fuir au moment opportun.
Différents dangers se mettront sur votre route: `Momie`, `Serpent`, `Scorpion`, `Araignée` et `Lave` attaquant les joueurs en jeu de façon aléatoire.

Pour lancer une partie, il faut exécuter le programme Jeu_DIAMOND.py.
Ensuite, une fenêtre python permet aux joueurs de définir leur nombre (entre 3 et 8), puis de choisir chacun leur tour leur pseudonyme.
Une fois cela fait, une interface graphique s'ouvrira, permettant aux joueurs de jouer simplement, en cliquant chacun leur tour soit sur
le bouton Poursuivre, soit sur le bouton Fuir. Toutes les actions qui se déroulent sont indiquées clairement pour que le joueur ne se perde pas
dans la partie.

	/!\ Remarques :

- Le programme ne prend pas en compte les égalités.
- Durant les chargements des manches et à l'affichage du classement final, le programme peut ralentir.
- Cliquer sur les zones des boutons de choix pendant les temps de chargement peut précipiter le choix du premier joueur à choisir.
- L'interface graphique ne permet pas d'entrer le nombre de joueurs ainsi que leur nom, cela se fait dans le terminal
- Le code ne permet de pas de relancer une partie, il faut rappeler le fichier Jeu_DIAMOND.py



------------ EN : ---------------!


The folder contains the files to run the program correctly.
The main program of the game is called `Jeu_DIAMOND.py`. The folder also contains a test file for certain functions, a folder containing all images required 
by the graphical interface, and a file. py named tkiteasy to run the graphical interface.

The purpose of this program is to play the game called DIAMOND in which players must choose at each turn between staying in the expedition or leaving.
As long as the players stay, rubies obtained during the round are not validated. They need to run away at the appropriate time.
Various dangers will come on your way: Mummy, Snake, Scorpion, Spider and Lava, which attack players in the game randomly.

To start a game, you beed to run the program Jeu_DIAMOND.py.
Then, a python window allows players to define their number (between 3 and 8), then to choose their own pseudo.
Once this is done, a graphical interface will open, allowing players to play simply, each clicking their turn either on
the Continue button, either on the Escape button. All actions that take place are clearly indicated so that the player does not get lost in the game.

/!\ Remarks :

- The program does not take into account equalities.
- During the loading of the rounds and the display of the final ranking, the program may slow down.
- Clicking on the selection button areas during loading times can precipitate the choice of the first player to choose.
- The graphical interface does not allow to enter the number of players as well as their name, this is done in the terminal.
- The code does not allow you to restart a part, you must recall the Jeu_DIAMOND file.py
